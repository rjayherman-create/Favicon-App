# Favicon Creator

Create a polished favicon kit for an app, website, or SaaS launch in minutes.

## Features

- Upload a logo or use brand initials
- Pick shape, colors, border, shadow, and style presets
- Preview browser tab, mobile home screen, sidebar, and tiny favicon sizes
- Export PNG sizes, `favicon.ico`, `apple-touch-icon.png`, and `site.webmanifest`
- Download a full ZIP kit with install instructions

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
